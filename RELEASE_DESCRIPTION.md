# 🧠 Smriti v3.5.0 — Production Ready

## Das Cognitive OS für OpenClaw Agents

Smriti ist ein vollständiges Betriebssystem für Agent-Intelligenz. Nicht nur ein Tool — ein Partner, der mit jeder Session intelligenter wird.

---

## 🎯 Was ist Smriti?

**Definition:** Ein selbstlernendes Cognitive OS mit 5 integrierten Systemen, das Agents von "stateless" zu "stateful" macht.

**Kernprinzip:** Nicht Robustheit. **Antifragilität.** — Jede Störung macht das System stärker.

---

## 🚀 Die 5 Systeme

### 1️⃣ Quality Engine (M1-M5 Metriken)
Misst Intelligenz über traditionelle Metriken hinaus:
- **M1** — Predictive Surprise (Neue Konzepte)
- **M2** — Denkraum-Erweiterung (Frameworks)
- **M3** — Paradigmen-Shift (Verständnisänderung)
- **M4** — Session Velocity (Iterationsgeschwindigkeit)
- **M5** — Anti-Fragile Resonanz (Produktive Kritik)

**Status:** Immer aktiv, automatisches Logging

---

### 2️⃣ Pattern Engine (Reflecty L1-L3)
Lernt aus jeder Session:
- **L1** — Beobachtet Verhaltensmuster
- **L2** — Findet Cross-Domain-Analogien
- **L3** — Sagt 7-Tage-Vorhersagen

**Status:** Hintergrund-Prozess, mem0-Integration mit File-Fallback

---

### 3️⃣ Improvement Engine (OUROBOROS)
Selbstverbesserung durch echte Mutationen:
- **Type A** — Parameter-Tuning (automatisch)
- **Type B** — Strukturelle Änderungen (mit Approval)
- **Type C** — Architektur-Änderungen (manuelle Review)

**Features:** Backup vor Mutationen, Rollback-Support, Signal-Erkennung

---

### 4️⃣ BRIDGE
Verbindet Pattern ↔ Improvement:
- Bidirektionales Event-Routing
- File-Locking (Race-Condition-sicher)
- Schema-Validation
- Conflict-Detection

---

### 5️⃣ Memory Infrastructure (L1-L4)
Vier Ebenen des Gedächtnisses:
- **L1** — Context Window (Session)
- **L2** — File System (immer verfügbar)
- **L3** — Semantic (mem0 + Qdrant, mit Fallback)
- **L4** — Graph (Neo4j, optional)

## ⚡ Quick Start

```bash
# 1. Download
git clone https://github.com/olisconsulting-cloud/Agent-Brain.git smriti
cd smriti

# 2. Install
node install.mjs

# 3. Auto-Setup (Fügt sich in AGENTS.md ein)
bash setup-auto.sh

# 4. Fertig! Nach Restart: Alles automatisch aktiv
```
